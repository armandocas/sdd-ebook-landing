# SDD Ebook - Landing Page

Landing page para o ebook **Spec Driven Development** de Armando Castellan Neto.

## 🚀 Tecnologias

- React 18
- Tailwind CSS
- Framer Motion (animações)
- Vite (build tool)

## 📦 Instalação Local

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`

## 🏗️ Build para Produção

```bash
npm run build
```

## 🌐 Deploy

O projeto está configurado para deploy automático na Netlify via GitHub.
